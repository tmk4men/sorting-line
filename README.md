# アルバイターじゅん

宅配の仕分けバイト体験から始まった、**ひと月を食いつなぐ生活ゲーム**。単一HTML・依存なし。

**▶ プレイ: https://tmk4men.github.io/sorting-line/**

じゅんの部屋で今日の出勤先を選び、ラインで荷物をさばいて日給を稼ぎ、
食費と家賃を払って翌日へ。30日を越えたときに手元にいくら残っているかが全て。

## 1日の流れ

1. **部屋**（ドット絵）で状態を確認して出勤先を選ぶ — 日数 / 時刻 / 所持金 / 残りシフト / 体力 / 体調 / アルバイターLv
2. **勤務**（ミニゲーム）— 長く続くほど処理数が伸びて給料は増えるが、そのぶん時計も進む
3. **給与明細** — 処理数 × 単価 ＋ コンボ手当 − ミス控除
4. **夜の過ごし方**を選ぶ — 食事と睡眠で翌日の体力・体調が決まる
5. **精算** — 食費（毎日）、家賃 ¥14,000（**7日ごと**）、光熱費（20日目）
6. 翌朝へ。所持金がマイナスになったらそこで終わり

## 出勤先

| 現場 | 単価 | 体力 | ミス上限 | ルール |
|---|---|---|---|---|
| 宅配 仕分けセンター | ¥135 / 個 | 38 | 3回 | 伝票の**担当番号**だけを抜く |
| 青果市場 選果ライン | ¥98 / 個 | 30 | 4回 | 黒い斑点が大きい**傷み品だけ**を弾く |
| クリーンセンター 分別ライン | **¥1,380 / 時**（時給制） | 46 | 3回 | 3本のレーンに資源が混ざって流れてくる。**つかんで上下にスワイプ**し、正しいレーン（1カン／2ビン／3ペット）へ移す。違うレーンのままカゴへ落ちると誤混入でミス（Lv3で解禁） |

単価・時給はアルバイターLvで上がる（Lvごとに +9%）。経験値は処理数・コンボ・到達レベルから入る。
分別ラインは**時給制**なので、長く粘るほど（＝時計を進めるほど）稼ぎが伸びる。個数制のバイトは処理数で決まる。

## 勤務パート（ミニゲーム）

- 作業員は**見えないマス目**の上を **十字キー（WASD可）** で移動する。1マスずつ。
- **目の前の1マスだけ**がうっすら黄色く光り、**そこにある荷物しか取れない**。
- **スペース／画面タップ／「取る」ボタン**で抜き取る。タッチ端末には十字キーUIが出る。
- 担当外を取る／担当を取りこぼすとミス。上限に達すると早上がり（給料も減る）。
- レベルが上がると速度・出現間隔・紛らわしさが増し、担当番号は**最大3つまで増える**（途中で入れ替わることはない）。
- 仕分けセンターでは**速達**（赤リボン）は2倍点、**重量物**（注意帯）は2回掴む。
- 23:00になると終電で強制的に上がり。

## 体力・体調・気分

- **体力**：出勤に必要。勤務時間が長いほど多く減る。夜の食事で回復。
- **体調**：食事と睡眠時間で上下する中長期ステータス。低いと回復が鈍り、**朝に寝込んで出勤できない日**が出る。14未満で倒れると病院代 ¥12,000。
- **気分**：放っておくと毎日じわじわ下がり、働くとさらに落ちる。**低いと勤務のミス上限が1回減り、体調も崩しやすくなる**。0になると燃え尽きて終了。
- 帰宅が21時／22時半を過ぎると睡眠不足で回復量が落ちる。寝込んでも家賃と食費は待ってくれない。

## 遊ぶ

部屋の「遊ぶ」から、気分を回復させる過ごし方を選ぶ。内容ごとに **お金・気分・体調・かかる時間** が違う。

| 遊び | 費用 | 効果 |
|---|---|---|
| 散歩する | 無料 | 気分 +12（60分） |
| ゲームセンター | ¥1,500 | 気分 +24（90分） |
| 居酒屋で一杯 | ¥3,200 | 気分 +34・体調 -6（150分） |
| スマホゲーに課金 | ¥5,000 | 当たれば気分大幅回復、外せば逆に下がる（40分・ギャンブル） |

遊ぶと時計が進むので、稼ぎとの兼ね合いになる。

## エンディング

30日を終えたときの所持金とLvで分岐（夜逃げ／まだ続く／貯金ができた／正社員登用）。所持金がマイナスで夜逃げ、気分が尽きると燃え尽きで途中終了。進行は `localStorage` に自動保存される。

## 実装メモ

- 単一HTML（Canvas 2D + Web Audio）。外部リソースなし、絵文字なし。図版はすべてコード内のベクター／ドット描画。
- 部屋は 320×200 のオフスクリーンcanvasにドット絵を描き、`imageSmoothingEnabled = false` で整数倍に拡大している。
- 画面幅で横長 / 縦長のレイアウトを切り替え、ライン速度も画面幅に合わせてスケールする。
- 効果音はオシレータとノイズバッファから生成。
- `?debug=1` で `window.__S` / `window.__LIFE` / `window.__start()` を公開（自動テスト用）。
