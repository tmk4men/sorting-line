# 仕分けレーン — SORTING LINE

宅配便の物流センターでの「仕分け」バイト体験をそのままゲームにした、単一HTMLのブラウザゲーム。

レーンを流れてくる荷物の伝票番号を読み、**自分の担当番号だけ**を抜き取る。
似た番号は混ざる、ラインは止まらない、そして担当は突然変わる。

**▶ プレイ: https://tmk4men.github.io/sorting-line/**

## 遊び方

| 操作 | 内容 |
|---|---|
| クリック／タップ | 荷物を抜き取る |
| `1` `2` `3` キー | 各レーンの先頭（取り出し口に一番近い）荷物を掴む |

- 担当番号の荷物を抜けば得点。**5個連続**ごとに倍率が上がり最大 x5。
- **他人の荷物を取る** / **担当の荷物を取りこぼす**（左端のシュートへ落とす）とミス。3回で終了。
- **速達**（赤いリボン）は2倍点。**重量物**（重マーク・大きい箱）は2回掴む必要がある。
- 12個さばくごとにレベルアップ。ライン速度・出現間隔・紛らわしい番号の比率が上がる。
- レベル4以降は担当が2つに増え、数レベルごとに**配置転換**で担当番号そのものが変わる。配置転換のときは、すでにライン上にある荷物も新しい担当で判定し直される。
- 終了時に SCORE / 処理数 / 最高コンボ / LEVEL / 正確率 / 誤仕分け / 取りこぼし / 平均反応時間 が出る。

## 実行

`index.html` をブラウザで開くだけ。ビルド不要・依存なし・オフライン動作。

## 実装メモ

- 単一HTML（Canvas 2D + Web Audio）。外部リソースなし、絵文字なし。図版はすべてベクター描画。
- 画面幅で横長 / 縦長のレイアウトを切り替え、速度も画面幅に合わせてスケールする。
- 効果音はオシレータとノイズバッファから生成。
- ハイスコアは `localStorage`（キー: `sortingline.best`）。
- `?debug=1` を付けると自動テスト用に内部状態を `window.__S` へ公開する。
